##如何安裝和運行
____
### 克隆存儲庫
 
```sh
$ cmd
$ git clone https://github.com/BEPb/Pornhub_dl_gui
$ cd Pornhub_dl_gui
```
 
### 安裝必要的包（安裝依賴）
```sh
$ pip install -r requirements.txt
```

＃＃ 用法
要啟動 GUI：
```
python3 -m pornhub_dl
```
