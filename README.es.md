## Cómo instalar y ejecutar
____
### Clonar el repositorio
 
```sh
$ cmd
$ git clone https://github.com/BEPb/Pornhub_dl_gui
$ cd Pornhub_dl_gui
```
 
### Instalar los paquetes necesarios (Instalar dependencias)
```sh
$  pip install -r requirements.txt
```

## Uso
Para iniciar la GUI:
```
python3 -m pornhub_dl
```
