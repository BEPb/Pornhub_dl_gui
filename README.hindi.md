## कैसे स्थापित करें और चलाएं
____
### रिपॉजिटरी को क्लोन करें
 
```sh
$ cmd
$ git clone https://github.com/BEPb/Pornhub_dl_gui
$ cd Pornhub_dl_gui
```
 
### आवश्यक पैकेज स्थापित करें (निर्भरता स्थापित करें)
```sh
$ pip install -r requirements.txt

```

## उपयोग
要啟動 GUI：
```
python3 -m pornhub_dl
```
