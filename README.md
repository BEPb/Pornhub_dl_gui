## How to install and run
____
### Clone the repository
 
```sh
$ cmd
$ git clone https://github.com/BEPb/Pornhub_dl_gui
$ cd Pornhub_dl_gui
```
 
### Install the necessary packages (Install dependencies)
```sh
$  pip install -r requirements.txt
```

## Usage
To launch the GUI:
```
python3 -m pornhub_dl
```
